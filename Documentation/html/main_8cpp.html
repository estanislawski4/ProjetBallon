Description détaillée

Programme principal pour la collecte et l'envoi de données météorologiques.

Ce programme utilise un capteur BME280 pour mesurer la température, la pression et l'humidité. Les données sont sauvegardées dans un fichier CSV toutes les 10 secondes et envoyées via une file IPC sous forme de trame APRS toutes les 2 minutes.
