Description détaillée

Classe permettant la gestion des communications I2C avec un périphérique.

Cette classe permet de lire et d'écrire des données avec un périphérique via le bus I2C. Elle prend en charge les transactions SMBus et fournit des méthodes pour interagir avec un esclave I2C.
