Detailed Description

Classe permettant de gérer la télémétrie d'un capteur BME280.

Cette classe obtient les mesures de température, pression et humidité à partir du capteur BME280, et les sauvegarde dans un fichier CSV au format approprié. Elle peut également générer une trame APRS.
