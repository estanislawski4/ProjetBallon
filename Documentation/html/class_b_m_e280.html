Detailed Description

Classe permettant d'interagir avec un capteur BME280 via I2C.

Classe pour interagir avec le capteur environnemental BME280.

Cette classe permet de communiquer avec le capteur BME280 en utilisant le bus I2C pour obtenir des mesures de température, de pression et d'humidité. Elle gère également la calibration du capteur et le calcul de la pression au niveau de la mer.
