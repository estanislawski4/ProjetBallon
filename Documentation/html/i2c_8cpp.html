Detailed Description

Classe pour le bus I2C \detail I2C (pronounce: I squared C) is a protocol developed by Philips. It is a slow two-wire protocol (variable speed, up to 400 kHz), with a high speed extension (3.4 MHz). It provides an inexpensive bus for connecting many types of devices with in frequent or low bandwidth communications needs. I2C is widely used with embedded systems.
