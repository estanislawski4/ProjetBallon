Detailed Description

Classe pour la gestion des files IPC.

Cette classe permet de gérer les files de messages IPC, permettant ainsi l'échange de données entre différents processus. Elle offre des méthodes pour créer une file IPC, écrire dans la file, et lire les messages à partir de la file.
